# YouTube Audio Pan & Boost

This userscript creates two new buttons after volume controls on the YouTube player bottom bar. The first one is a `volume multiplier` which increase the gain on top of the actual volume slider, clicking it will toggle a slider to modify the value. The second one is the `Pan` button which will toggle a slider to pan the audio either left or right. When hiding the pan slider the pan setting will be reset to center.
